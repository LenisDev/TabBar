# TabBar
Easy to use functional tab bar for swift projects

### Installation Guide

#### CocoaPods

Step 1:
Add pod 'Tab', '~> 0.0' to your Podfile:

```
target 'MyApp' do
  pod 'Tab', '~> 0.0'
end
```

Step 2:
run `pod install` from current project directory.

*That's all, you have it installed.*

<br>

>
>
>Try it using following command
>```
>pod try Tab
>```
